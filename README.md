# Stock-Management-System

## Composer install
```
composer install
```

## Laravel init
```
php artisan key:generate
php artisan migrate
php artisan db:seed
```

## serve at localhost:8000
```
php artisan serve
```

## serve at specific port
```
php artisan serve --port=8001
```

## serve with hot reload at localhost:3000
```
npm run watch
```
